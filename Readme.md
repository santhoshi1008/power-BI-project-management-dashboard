# Project Management Dashboard - Power BI

## Overview
Developed an interactive Project Management Dashboard in Power BI to monitor project performance, budgets, project status, and department-wise analytics.

## Tools Used
- Power BI
- Power Query
- DAX
- Microsoft Excel

## Key Features
- KPI Cards for Total Projects, Total Budget, Completed Projects, In Progress Projects, and Average Duration.
- Department-wise project analysis.
- Budget analysis by manager and department.
- Project status distribution.
- Interactive slicers for Department, Manager, Status, and Date.

## DAX Measures
- Total Projects
- Total Budget
- Completed Projects
- In Progress Projects
- Average Duration

## Dataset
Synthetic project management dataset containing:
- Project ID
- Project Name
- Department
- Manager
- Start Date
- End Date
- Budget
- Status

## Insights
- Finance department has the highest budget allocation.
- Completed and In Progress projects are nearly balanced.
- Average project duration is approximately 108 days.
